# MachineLearning23

Afleveringer/Projekter for kruset "Introduktion til Machine Learning" i efteråret 2023 i samarbejde med Alexander Myrtoft Skjødt. 

Koden mangler for det tredje projekt, da den er blevet væk.

Projekt 1:
- Implementering af logistisk regression
- Implementering af softmax regression
Projekt 2:
- Implementering af et neural netværk med et enkelt skjult lag
Projekt 3:
- Semisupervised deep learning
